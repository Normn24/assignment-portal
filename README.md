# Assignment Portal

## Running the Project Locally

Follow these steps to run the project locally:
1. **Install the dependencies**:
    npm install

2. **Build the app**:
    npm run build

3. **Start the development server:**:
    npm run start

## Running the Tests    
1. **Run the tests**:
    npm run test

2. **Run tests with watch mode (optional):**:    
    npm run test:watch
